# Introduction
This api provides data about records and their users.
# Installation

1. Clone repository
```bash
$ git clone https://github.com/aknrdlt/zan-talaby
$ cd zan-talaby
```
2. Install dependencies
```bash
$ composer update

$ composer install

$ php artisan passport:install

$ php artisan migrate
```
3. Run server
```bash
$ php artisan serve
```
